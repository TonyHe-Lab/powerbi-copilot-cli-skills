# Project Brief

## Project

Power BI Copilot CLI Skills

## Purpose

Create a focused reusable GitHub Copilot CLI skill project that distills the most useful Power BI guidance from the upstream Data Goblin marketplace into a smaller, easier-to-install plugin.

## In Scope

- semantic modeling guidance
- DAX writing guidance
- Power Query authoring guidance
- report design guidance
- PBIP and TMDL management guidance
- local validation and documentation

## Out of Scope

- full upstream marketplace parity
- Fabric admin and tenant management
- custom agents, hooks, or MCP automation
- every upstream reference file
