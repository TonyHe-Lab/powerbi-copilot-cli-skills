# Product Context

## Why this exists

The upstream repository is valuable but broad. This refactor gives one analyst or engineer a concentrated Power BI skill pack for repeated Copilot CLI use without needing the whole marketplace.

## User outcome

The user should be able to publish this folder as a GitHub repo and reuse a tight Power BI skill set in Copilot CLI for common analysis and model-authoring tasks.

## UX goals

- small number of skills with obvious names
- fast discovery from `Use when...` descriptions
- concise content that is practical during real work
